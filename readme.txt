== Description ==

Add a really simple sitemap to your blog.  All posts are only 2 clicks away from the home page.
Using this plugin also makes it really easy for search engines to spider and index all of your blog.


== Installation ==

1. Upload `rs-sitemap.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the options in the RS Sitemap options admin page
4. Create a page and add the text `<!--rs-sitemap-->` to it

== Frequently Asked Questions ==

= Do I need to add the `<!--rs-sitemap-->` to a Post or a Page? =

You can add it to either.  The text will be substituted with the sitemap links.

= What should I call the page that I add the sitemap to? =

You can call it whatever you like.  I would suggest you call it `Site Map`.
